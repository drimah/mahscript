# MahScript Pro v3.0

Plugin Chrome para Google Docs com **11 skins visuais** e **sistema de capítulos com drag-and-drop**.

## Como instalar

1. Abra o Chrome e vá em `chrome://extensions`
2. Ative **"Modo do desenvolvedor"** (canto superior direito)
3. Clique em **"Carregar sem compactação"**
4. Selecione a pasta `mahscript-plugin`
5. Abra qualquer documento em `docs.google.com`

## Como usar

- **Botão roxo** (canto inferior direito) abre o menu
- **Skins & Temas** → escolha entre 11 paletas de cores
- **Capítulos** → abre sidebar com todos os títulos H1 do documento

### Sidebar de Capítulos
- **Clique** no capítulo para navegar até ele no documento
- **Arraste** (ícone de grip) para reordenar os capítulos
- **Ícone de edição** para renomear
- **Ícone de lixeira** para excluir (remove o texto do doc!)
- **Busca** filtra capítulos em tempo real

### Criar Capítulo
1. Abra a sidebar → "Novo Capítulo"
2. Digite o nome e um texto inicial (opcional)
3. O capítulo é inserido como Título H1 no final do documento
